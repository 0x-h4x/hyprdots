# minimal monochrome rice for hyprland


### minimal hyprland rice made for notebooks.

Shortcuts:
```Super + F - FireFox

Super + Q - Kitty

Super + E - PcmanFM-QT

Super + R - Wofi

Super + ESC - Nwg-Bar
PRT SCR - hyprshot -m region

(See more in hypr/hyprland.conf)
```

Packages:
```
Wofi
PcmanFM
Kitty
Waybar
NWG-Bar
hyprshot
hyprlock
brightnessctl
neofetch

```


No code is original, just modified to fit my use.
